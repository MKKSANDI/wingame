  ⚙️ CPU OPTIMIZATIONS (8 Total)
═══════════════════════════════════════════════════════════════════════

1. DISABLE POWER THROTTLING
   Registry: HKLM\SOFTWARE\Policies\Microsoft\Windows\System
   Effect: Removes Windows CPU performance limits
   Impact: 5-15% CPU performance gain
   Restart: No

2. OPTIMIZE CPU FOR GAMING
   Registry: HKLM\SOFTWARE\...\Multimedia\SystemProfile\Tasks\Games
   Values Changed:
     • GPU Priority = 8
     • Priority = 6 (High)
     • Scheduling Category = High
     • SFIO Priority = High
     • Clock Rate = 10000
     • Affinity = 0
   Effect: Games get highest CPU priority
   Impact: Reduced stuttering, smoother frame times
   Restart: No

3. INCREASE PC RESPONSIVENESS
   Registry: HKLM\SOFTWARE\...\Multimedia\SystemProfile
   Values Changed:
     • NetworkThrottlingIndex = 0xFFFFFFFF (disabled)
     • SystemResponsiveness = 0 (instant)
   Effect: Zero network throttling, instant system response
   Impact: Lower input lag, faster network
   Restart: No

4. DISABLE GAME DVR & GAME BAR
   Registry: HKCU\System\GameConfigStore
   Values Changed:
     • GameDVR_Enabled = 0
     • GameDVR_FSEBehaviorMode = 2
     • Additional DVR settings disabled
   Effect: Disables Xbox Game Bar recording
   Impact: 3-10 FPS gain, less CPU/GPU usage
   Restart: No

5. DISABLE AUTOMATIC MAINTENANCE
   Registry: HKLM\SOFTWARE\...\Schedule\Maintenance
   Value: MaintenanceDisabled = 1
   Effect: Prevents Windows maintenance during gaming
   Impact: No background interruptions
   Restart: No

6. DISABLE ENERGY ESTIMATION
   Registry: HKLM\SYSTEM\...\Control\Power
   Service: DiagTrack (stopped and disabled)
   Value: EnergyEstimationEnabled = 0
   Effect: Stops energy logging service
   Impact: Less CPU overhead
   Restart: No

7. ENABLE CPU PRIORITY (Vendor-Specific)
   
   FOR AMD:
     Registry: HKLM\SOFTWARE\...\Tasks\Games
     Values: Priority=6, Scheduling=High, GPU Priority=8
   
   FOR INTEL:
     Registry: HKLM\SOFTWARE\...\Tasks\Games
     Additional: HKLM\SOFTWARE\Intel\GMM
     Values: DedicatedSegmentSize=1298
   
   Effect: CPU-specific optimizations
   Impact: Better thread scheduling
   Restart: No

8. USE ALL CPU CORES AT STARTUP
   Command: bcdedit /set {current} numproc [CPU_COUNT]
   Effect: Forces Windows to use all cores during boot
   Impact: Faster boot, better core utilization
   Restart: Yes (required)

  🎮 GPU OPTIMIZATIONS (3 Total)
═══════════════════════════════════════════════════════════════════════

1. GAMING GPU OPTIMIZATIONS
   Registry: HKLM\SOFTWARE\...\Tasks\Games
   Same as CPU Gaming Optimization
   Effect: Optimizes multimedia profile for gaming
   Impact: Better GPU scheduling
   Restart: No

2. GPU PERFORMANCE TIER (Dropdown Selection)
   
   LOW-MEDIUM (Budget GPU):
     Registry: SystemResponsiveness = 1
     Best for: GTX 1050, RX 560, integrated GPUs
   
   MEDIUM-HIGH (Powerful GPU):
     Registry: SystemResponsiveness = 0
     Best for: RTX 3060+, RX 6700+, high-end GPUs
   
   Effect: Adjusts system priorities for GPU capability
   Impact: Balanced performance
   Restart: No

3. ENABLE GPU THREAD PRIORITY (Vendor-Specific)
   
   FOR NVIDIA:
     Registry: HKLM\SYSTEM\...\nvlddmkm\Parameters
     Value: ThreadPriority = 31 (highest)
   
   FOR AMD:
     Registry: HKLM\SYSTEM\...\amdkmdap\Parameters
     Value: ThreadPriority = 31 (highest)
   
   FOR INTEL:
     Registry: HKLM\SOFTWARE\Intel\GMM
     Value: DedicatedSegmentSize = 1298
   
   Effect: GPU driver threads get highest priority
   Impact: Better frame times, less stuttering
   Restart: No

  💻 SYSTEM OPTIMIZATIONS (3 Total)
═══════════════════════════════════════════════════════════════════════

1. DISABLE HYPER-V VIRTUALIZATION
   Command: dism /online /disable-feature /featurename:Microsoft-Hyper-V-All
   Effect: Disables Windows virtualization features
   Impact: Frees CPU resources, better performance
   Restart: Yes (recommended)

2. DISABLE VIRTUAL MACHINE PLATFORM
   Command: dism /online /disable-feature /featurename:VirtualMachinePlatform
   Effect: Disables VM platform (WSL2 backend)
   Impact: Reduces CPU overhead
   Restart: Yes (recommended)

3. DISABLE UNNECESSARY BACKGROUND SERVICES
   Services Stopped:
     • SysMain (Superfetch) - disk caching
     • WSearch (Windows Search) - file indexing
     • DiagTrack (Diagnostics) - telemetry
   
   Commands: 
     sc config [service] start=disabled
     sc stop [service]
   
   Effect: Stops resource-heavy background services
   Impact: More CPU/RAM available for games
   Restart: No (but recommended)
